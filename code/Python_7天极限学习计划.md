# Python 7天极限速成计划

> 本计划基于《Python 完整学习笔记》及《高效学习推荐法》深度压缩，适合有一定编程基础或希望高强度突击学习的开发者。

---

## 🧭 极限学习生存法则

时间紧迫，请必须严格执行以下 **三大核心策略**：
1. **「拆注释 → 自己写」**：不要从零敲代码！直接看笔记中的代码，取消注释跑一遍，理解后盖住原代码默写。
2. **「一章一卡」**：不用记所有细节，每章只总结一张包含“核心语法+最小示例”的知识卡片，随用随查。
3. **「小项目驱动」**：光看不练等于没学。每天收尾时，必须把当天知识融合成一个闭环小项目。

---

## 🗓️ 7天极限突击时间表

### 🟩 第1天：地基与逻辑控制（对应：第1-4章）
**目标**：掌握基本数据结构和循环分支，能够写出基础逻辑。
* **上午**：【第1-3章】环境搭建、变量、数据类型（整型、浮点型、布尔型）、字符串格式化（`f-string`）、各种运算符。
* **下午**：【第4章】流程控制：`if-elif-else` 分支判断；`while` 和 `for` 循环；`continue` 和 `break` 控制。
* **晚上（实战）**：🏆 **答题闯关挑战赛**（结合输入 `input()`、`while` 循环、`if` 判断）。默写**九九乘法表**。

### 🟨 第2天：数据与功能封装（对应：第5-6章）
**目标**：精通四大容器的增删改查，掌握函数的高效复用。
* **上午**：【第6章】数据容器大决战：熟练掌握 **列表(List)** 和 **字典(Dict)** 的增删改查及循环遍历；了解元组、集合和字符串的常用操作（切片等）。
* **下午**：【第5章】函数基础：定义、调用、各类参数（位置、关键字、默认值、不定长 `*args`/`**kwargs`）、返回值、变量作用域（局部/全局）。
* **晚上（实战）**：🏆 **学生成绩/信息管理系统**（利用字典和列表存储数据，封装成多个函数模块进行增删改查）。

### 🟧 第3天：面向对象编程 OOP（对应：第7章）
**目标**：跨越新手村，建立对象思维，理解代码是如何在大型工程中被组织的。
* **上午**：【第7章】类与对象基础：定义类、`__init__` 初始化、实例属性与实例方法、类属性与类方法（`@classmethod`）。
* **下午**：【第7章】三大特性与魔法方法：单继承/多重继承、方法重写、`super()`；了解多态（鸭子类型）与常见的魔法方法（`__str__`, `__eq__`等）。
* **晚上（实战）**：🏆 **面向对象版学生管理系统**（将昨天的函数版重构为基于 `Manager` 类和 `Student` 类的 OOP 版本）。

### 🟥 第4天：进阶语法与Python灵魂（对应：第8章）
**目标**：写出具有“Python 味”的优雅代码，彻底搞懂闭包与装饰器。
* **上午**：【第8章】高阶特性：匿名函数 `lambda`、条件表达式、四大金刚函数（`map/filter/sorted/reduce`）、列表推导式。
* **下午**：【第8章】闭包与装饰器：理解四种作用域（`nonlocal`）；彻底弄懂闭包原理；掌握函数装饰器的写法与运行机制。
* **晚上（实战）**：🏆 **函数功能增强器**（写一个通用的装饰器，能够在不修改原函数代码的前提下，打印函数的执行耗时及日志信息）。

### 🟪 第5天：异常处理、模块与持久化（对应：第9、10、12章）
**目标**：让程序健壮且能保存数据，能引用外部工具。
* **上午**：【第9-10章】异常处理（`try-except-finally`）与主动抛出异常（`raise`）；模块与包的导入规范（`import`），`__name__`的妙用。
* **下午**：【第12章】文件读写：使用 `with open()` 操作纯文本文件（r/w/a 模式）；了解基础的 `os` 目录操作。
* **晚上（实战）**：🏆 **基于文件的持久化登录系统**（实现带有异常校验的登录功能，并将成功/失败日志追加写入 `log.txt` 文件）。

### 🟦 第6天：迭代器与生成器（对应：第11章）
**目标**：掌握惰性计算思想，能够处理极大规模的数据而不撑爆内存。
* **上午**：【第11章】迭代器：理解可迭代对象与迭代器协议（`__iter__`, `__next__`），理解 `for` 循环底层的运行机制。
* **下午**：【第11章】生成器：掌握生成器函数（`yield` 的使用、暂停与传值 `send`）、生成器表达式。
* **晚上（实战）**：🏆 **惰性数据生成器**（尝试手写一个类迭代器，或者使用 `yield` 实现一个能生成十万级斐波那契数列但内存占用极小的生成器）。

### 🟫 第7天：多任务并发编程（对应：第13章）
**目标**：利用计算机多核/多线程优势，大幅提升代码执行效率。
* **上午**：【第13章】多进程与多线程基础：`Process` 和 `Thread` 的基本用法、`join` 和 `daemon` 守护机制；理解 GIL 锁和 CPU/IO 密集型任务的区别。
* **下午**：【第13章】并发控制与进阶：互斥锁（`Lock/RLock`）、进程间通信（`Queue`）；掌握进程池/线程池（`ThreadPoolExecutor`）的高效批量派发与回收。
* **晚上（实战）**：🏆 **多线程批量文件复制器**（模拟 IO 密集型任务，对比单线程与多线程池在处理文件拷贝上的速度差异）。

## 📌 极限突击建议
* **只抓核心**：这7天内，如果遇到某些冷门特性（如 `__mro__` 的复杂优先级、不常用的标准库等），如果暂时无法理解，**立刻跳过**。先保证核心的主线跑通。
* **每天复盘**：晚上睡觉前，闭上眼睛把今天学的大知识块在大脑里像放电影一样过一遍，第二天效率会翻倍。
