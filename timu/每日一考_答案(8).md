### 题目 1：文件操作 - 读取文件内容

假设有一个文本文件 test.txt，内容为 "Hello, World!"，编写 Python 代码读取该文件的内容并打印出来。

**答案**：

```python
try:
    with open('test.txt', 'r', encoding='utf - 8') as file:
        content = file.read()
        print(content)
except FileNotFoundError:
    print("文件未找到")
```

**分析**：使用 with open 语句打开文件，'r' 表示以只读模式打开，encoding='utf - 8' 确保能正确读取文本文件中的字符。file.read() 方法读取文件的全部内容并赋值给 content，最后打印 content。try - except 块用于捕获可能出现的 FileNotFoundError 异常，避免程序因文件不存在而崩溃。

### 题目 2：文件操作 - 写入文件

编写代码创建一个新的文本文件 output.txt，并向其中写入字符串 "This is a test."。

**答案**：

```python
with open('output.txt', 'w', encoding='utf - 8') as file:
    file.write("This is a test.")
```

**分析**：以 'w' 写入模式打开 output.txt 文件，如果文件不存在则创建新文件。file.write() 方法将指定字符串写入文件中。with open 语句在代码块结束后会自动关闭文件，确保资源正确释放。

### 题目 3：面向对象 - 类与对象基础

定义一个名为 Dog 的类，该类有两个属性 name（名字）和 age（年龄），以及一个方法 bark（叫），bark 方法打印出 "Woof! My name is [name] and I am [age] years old."。创建一个 Dog 类的对象，并调用 bark 方法。

**答案**：

```python
class Dog:
    def __init__(self, name, age):
        self.name = name
        self.age = age
    def bark(self):
        print(f"Woof! My name is {self.name} and I am {self.age} years old.")
my_dog = Dog("Buddy", 3)
my_dog.bark()
```

**分析**：在 Dog 类中，__init__ 方法是构造函数，用于初始化对象的属性 name 和 age。self 代表对象本身，通过它可以访问和设置对象的属性。bark 方法使用 self.name 和 self.age 打印出相应信息。创建 Dog 类的对象 my_dog 时，传入 "Buddy" 和 3 作为 name 和 age 的值，然后调用 my_dog.bark() 方法执行打印操作。

### 题目 4：面向对象 - 对象属性访问

基于上一题的 Dog 类，创建一个 Dog 对象，然后修改其 age 属性的值，并再次调用 bark 方法查看输出变化。

**答案**：

```python
class Dog:
    def __init__(self, name, age):
        self.name = name
        self.age = age
    def bark(self):
        print(f"Woof! My name is {self.name} and I am {self.age} years old.")
my_dog = Dog("Max", 5)
my_dog.age = 6
my_dog.bark()
```

**分析**：首先创建 Dog 对象 my_dog，初始 age 为 5。通过 my_dog.age = 6 语句直接修改对象的 age 属性值。再次调用 my_dog.bark() 方法时，由于 age 属性已被修改，打印信息中的年龄也随之改变，体现了对象属性的可变性和动态访问特性。
