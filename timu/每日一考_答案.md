### 题目 1：分支与循环结合

编写一个程序，从 1 循环到 10，当数字是偶数时打印 “偶数”，奇数时打印 “奇数”。

```python
for num in range(1, 11):
    if num % 2 == 0:
        print("偶数")
    else:
        print("奇数")
```

**答案**：for循环遍历 1 到 10 的数字，在每次循环中，通过if-else分支语句判断数字是奇数还是偶数，并打印相应结果。

**分析**：该题综合考查分支语句和循环语句的结合使用，让学生明白如何在循环过程中根据条件执行不同操作，提升代码综合运用能力。

### 题目 2：循环控制

使用for循环，打印如下图形的*

```python
    *
   ***
  *****
 *******
*********
# 定义等腰三角形的行数
n = 5
for i in range(1,n + 1):
    # 打印每一行前面的空格
    for j in range(n - i):
        print(' ', end='')
    # 打印每一行的星号
    for k in range(2 * i - 1):
        print('*', end='')
    print()  # 换行

```

### 题目 3：列表操作

现有列表 my_list = [10, 20, 30, 40, 50]，请编写代码实现：

1. 向列表末尾添加一个元素 60。

1. 取出列表中索引为 2 的元素。

1. 计算列表中所有元素的和。

**答案**：

```python
my_list = [10, 20, 30, 40, 50]
# 向列表末尾添加元素60
my_list.append(60)
# 取出索引为2的元素
element = my_list[2]
# 计算列表元素和
sum_value = sum(my_list)
print(my_list)
print(element)
print(sum_value)  
```

**分析**：

1. append 方法用于在列表末尾添加元素，这是列表常用的修改操作。

1. 通过索引 [2] 可以准确访问列表中的元素，索引从 0 开始。

1. sum 函数可快速计算列表中所有数值元素的总和，体现了 Python 内置函数对列表操作的便利性。
